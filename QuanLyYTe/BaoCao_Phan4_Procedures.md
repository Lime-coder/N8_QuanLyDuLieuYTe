## PHỤ LỤC: DANH SÁCH CÁC PROCEDURE (THỦ TỤC LƯU TRỮ) SỬ DỤNG TRONG DỰ ÁN

Để đảm bảo nguyên tắc đặc quyền tối thiểu (Principle of Least Privilege) và ngăn chặn SQL Injection, toàn bộ các thao tác nghiệp vụ của ứng dụng đối với Cơ sở dữ liệu đều được thực hiện thông qua các Stored Procedures thay vì chạy lệnh SQL thô (Raw SQL). Dưới đây là tập hợp các Procedures chính yếu được xây dựng trong dự án.

### 1. Nhóm Procedure Quản trị Tài khoản và Phân quyền (Account & Role Management)
Các thủ tục này hỗ trợ DBA hoặc hệ thống quản trị người dùng, thao tác chủ yếu trên các view DBA (DBA_USERS, DBA_ROLES).

```sql
-- Lấy danh sách toàn bộ Users
CREATE OR REPLACE PROCEDURE USP_GET_ALL_USERS (p_cursor OUT SYS_REFCURSOR) AUTHID CURRENT_USER AS ...

-- Lấy danh sách toàn bộ Roles
CREATE OR REPLACE PROCEDURE USP_GET_ALL_ROLES (p_cursor OUT SYS_REFCURSOR) AUTHID CURRENT_USER AS ...

-- Tạo User mới và cấp quyền đăng nhập cơ bản
CREATE OR REPLACE PROCEDURE USP_CREATE_USER (p_username IN VARCHAR2, p_password IN VARCHAR2) AUTHID CURRENT_USER AS ...

-- Cập nhật mật khẩu User
CREATE OR REPLACE PROCEDURE USP_UPDATE_USER_PASSWORD (p_username IN VARCHAR2, p_new_password IN VARCHAR2) AUTHID CURRENT_USER AS ...

-- Khóa/Mở khóa User (Đồng thời đồng bộ trạng thái is_active vào bảng staff/patient)
CREATE OR REPLACE PROCEDURE USP_LOCK_USER (p_username IN VARCHAR2) AUTHID CURRENT_USER AS ...
CREATE OR REPLACE PROCEDURE USP_UNLOCK_USER (p_username IN VARCHAR2) AUTHID CURRENT_USER AS ...

-- Xóa User
CREATE OR REPLACE PROCEDURE USP_DROP_USER (p_username IN VARCHAR2, p_cascade  IN VARCHAR2 DEFAULT 'YES') AUTHID CURRENT_USER AS ...

-- Quản lý Role (Tạo, Sửa mật khẩu, Xóa)
CREATE OR REPLACE PROCEDURE USP_CREATE_ROLE (p_role_name IN VARCHAR2, p_password IN VARCHAR2 DEFAULT NULL) AUTHID CURRENT_USER AS ...
CREATE OR REPLACE PROCEDURE USP_UPDATE_ROLE_PASSWORD (p_role_name IN VARCHAR2, p_password IN VARCHAR2 DEFAULT NULL) AUTHID CURRENT_USER AS ...
CREATE OR REPLACE PROCEDURE USP_DROP_ROLE (p_role_name IN VARCHAR2) AUTHID CURRENT_USER AS ...
```

### 2. Nhóm Procedure Hỗ trợ Xác thực (Authentication & Identification)
Dùng để xác định ID nhân viên/bệnh nhân hoặc lấy role của người dùng ngay khi họ vừa đăng nhập vào hệ thống.

```sql
-- Lấy role đã được cấp cho một User
CREATE OR REPLACE PROCEDURE USP_GET_GRANTED_ROLE (p_user IN VARCHAR2, p_cursor OUT SYS_REFCURSOR) AUTHID CURRENT_USER AS ...

-- Lấy role đang active trong Session hiện tại
CREATE OR REPLACE PROCEDURE USP_GET_SESSION_ROLE (p_cursor OUT SYS_REFCURSOR) AUTHID CURRENT_USER AS ...

-- Mapping từ username_db sang staff_id hoặc patient_id thực tế
CREATE OR REPLACE PROCEDURE USP_GET_USER_ID (p_username IN VARCHAR2, p_role IN VARCHAR2, p_cursor OUT SYS_REFCURSOR) AUTHID DEFINER AS ...
```

### 3. Nhóm Procedure Nghiệp vụ Bác sĩ (Doctor Workflows)
Các thao tác của bác sĩ được phân tách riêng biệt, kết hợp với các chính sách VPD ngầm bên dưới.

```sql
-- Tra cứu và cập nhật hồ sơ bệnh án
CREATE OR REPLACE PROCEDURE USP_GET_MEDICAL_RECORD(p_s NVARCHAR2, p_c OUT SYS_REFCURSOR) AS ...
CREATE OR REPLACE PROCEDURE USP_UPDATE_MEDICAL_RECORD(p_id VARCHAR2, p_dg NVARCHAR2, p_tr NVARCHAR2, p_cl NVARCHAR2) AS ...

-- Quản lý dịch vụ cận lâm sàng (Chỉ định dịch vụ mới, xóa dịch vụ chưa thực hiện)
CREATE OR REPLACE PROCEDURE USP_GET_SERVICES(p_s NVARCHAR2, p_c OUT SYS_REFCURSOR) AS ...
CREATE OR REPLACE PROCEDURE USP_ADD_SERVICE(p_id VARCHAR2, p_type NVARCHAR2) AS ...
CREATE OR REPLACE PROCEDURE USP_DELETE_SERVICE(p_id VARCHAR2, p_type NVARCHAR2, p_date DATE) AS ...

-- Quản lý đơn thuốc (Kê đơn, sửa đổi, hủy)
CREATE OR REPLACE PROCEDURE USP_GET_PRESCRIPTION(p_s NVARCHAR2, p_c OUT SYS_REFCURSOR) AS ...
CREATE OR REPLACE PROCEDURE USP_MANAGE_PRESCRIPTION(p_action VARCHAR2, p_record_id VARCHAR2, p_med_name NVARCHAR2, p_dosage NVARCHAR2, p_date DATE DEFAULT NULL, p_old_med_name NVARCHAR2 DEFAULT NULL) AS ...

-- Xem danh sách và cập nhật tiền sử của Bệnh nhân
CREATE OR REPLACE PROCEDURE USP_GET_PATIENTS(p_s NVARCHAR2, p_c OUT SYS_REFCURSOR) AS ...
CREATE OR REPLACE PROCEDURE USP_UPDATE_PATIENT(p_id VARCHAR2, p_history NCLOB, p_family_history NCLOB, p_allergy NCLOB) AS ...

-- Bác sĩ tự xem và cập nhật thông tin cá nhân (SĐT, Quê quán)
CREATE OR REPLACE PROCEDURE USP_GET_SELF_INFO(p_c OUT SYS_REFCURSOR) AS ...
CREATE OR REPLACE PROCEDURE USP_UPDATE_SELF_INFO(p_hometown NVARCHAR2, p_phone VARCHAR2) AS ...
```

### 4. Nhóm Procedure Nghiệp vụ Điều phối viên (Coordinator Workflows)
Bao gồm các thao tác quản lý nhân sự chung và phân bổ bệnh nhân cho bác sĩ.

```sql
-- Lấy danh sách nhân sự (Bác sĩ, Kỹ thuật viên)
CREATE OR REPLACE PROCEDURE SP_COORD_GET_DOCTORS(p_cursor OUT SYS_REFCURSOR) AS ...
CREATE OR REPLACE PROCEDURE SP_COORD_GET_TECHS(p_cursor OUT SYS_REFCURSOR) AS ...
CREATE OR REPLACE PROCEDURE SP_COORD_GET_DEPTS(p_cursor OUT SYS_REFCURSOR) AS ...

-- Lọc bác sĩ theo chuyên khoa
CREATE OR REPLACE PROCEDURE SP_COORD_GET_DOC_DEPT(p_dept_id IN VARCHAR2, p_cursor OUT SYS_REFCURSOR) AS ...

-- Điều phối viên tự cập nhật thông tin cá nhân
CREATE OR REPLACE PROCEDURE SP_COORD_GET_SELF(p_cursor OUT SYS_REFCURSOR) AS ...
CREATE OR REPLACE PROCEDURE SP_COORD_UPD_SELF(p_phone IN VARCHAR2, p_hometown IN NVARCHAR2) AS ...

-- Quản lý hành chính Bệnh nhân (CRUD thông tin cá nhân)
CREATE OR REPLACE PROCEDURE SP_COORD_GET_PATS(p_cursor OUT SYS_REFCURSOR) AS ...
CREATE OR REPLACE PROCEDURE SP_COORD_SEARCH_PATS(p_keyword IN NVARCHAR2, p_cursor OUT SYS_REFCURSOR) AS ...
CREATE OR REPLACE PROCEDURE SP_COORD_INS_PAT(...) AS ...
CREATE OR REPLACE PROCEDURE SP_COORD_UPD_PAT(...) AS ...

-- Kiểm tra trùng lặp thông tin (ID, CCCD, Username)
CREATE OR REPLACE PROCEDURE SP_COORD_CHK_PAT_ID(p_patient_id IN VARCHAR2, p_count OUT NUMBER) AS ...
CREATE OR REPLACE PROCEDURE SP_COORD_CHK_IDCARD(p_id_card IN VARCHAR2, p_exclude_id IN VARCHAR2, p_count OUT NUMBER) AS ...
CREATE OR REPLACE PROCEDURE SP_COORD_CHK_USER(p_username IN VARCHAR2, p_exclude_id IN VARCHAR2, p_count OUT NUMBER) AS ...

-- Quản lý quy trình khám (Tạo bệnh án sơ bộ, phân công bác sĩ, phân công kỹ thuật viên)
CREATE OR REPLACE PROCEDURE SP_COORD_GET_ALL_MED(p_cursor OUT SYS_REFCURSOR) AS ...
CREATE OR REPLACE PROCEDURE SP_COORD_INS_MED(p_record_id IN VARCHAR2, p_patient_id IN VARCHAR2, p_record_date IN DATE, p_doctor_id IN VARCHAR2, p_dept_id IN VARCHAR2) AS ...
CREATE OR REPLACE PROCEDURE SP_COORD_UPD_MED_REC(p_record_id IN VARCHAR2, p_doctor_id IN VARCHAR2, p_dept_id IN VARCHAR2) AS ...
CREATE OR REPLACE PROCEDURE SP_COORD_GET_SRV_ASS(p_cursor OUT SYS_REFCURSOR) AS ...
CREATE OR REPLACE PROCEDURE SP_COORD_UPD_TECH(p_record_id IN VARCHAR2, p_service_type IN NVARCHAR2, p_service_date IN DATE, p_technician_id IN VARCHAR2) AS ...

-- Phân trang (Pagination) xử lý danh sách lớn
CREATE OR REPLACE PROCEDURE SP_COORD_GET_PATS_PAGED(...) AS ...
CREATE OR REPLACE PROCEDURE SP_COORD_GET_ALL_MED_PAGED(...) AS ...
CREATE OR REPLACE PROCEDURE SP_COORD_GET_SRV_ASS_PAGED(...) AS ...
```

*(Lưu ý: Để giữ độ dài tài liệu hợp lý, phần thân chi tiết của một số Procedure đã được lược gọn, thay vào đó tập trung giải thích cấu trúc và công dụng của từng nhóm Procedure).*
